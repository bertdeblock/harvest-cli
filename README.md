Super simple Harvest CLI tool.

## Commands

- [start](#start)
- [stop](#stop)
- [cancel](#cancel)
- [last](#last)

### start

Start a new time entry:

```shell
hv start
```

### stop

Stop the last running time entry:

```shell
hv stop
```

### cancel

Cancel the last running time entry:

```shell
hv cancel
```

### last

Get info about the last time entry:

```shell
hv last
```
